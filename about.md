# Super Hack Installer

A simple installer mod for Geometry Dash 2.2081 using Geode 5.6.1.

## Features

* Easy access from the main menu
* Clean and lightweight UI
* Install / uninstall supported menus
* Built for Geode 5.x
* Fast and simple workflow

## Requirements

* Geometry Dash 2.2081
* Geode 5.6.1 or newer

## Installation

1. Download the latest release.
2. Place the `.geode` file into your Geode mods folder.
3. Launch Geometry Dash.
4. Open the installer from the main menu.
